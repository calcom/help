---
title: "Guest notifications"
description: "Control which booking-related emails your organization sends to guests."
---

Organization admins can manage guest notification emails from **Settings > Organization > Guest notifications**. These settings apply to all event types across your organization.

<Warning>
  Disabling guest emails means attendees will not receive important booking updates. If you turn off emails, consider setting up [org-wide workflows](/workflows/workflowsoverview) to keep your guests informed.
</Warning>

## Disable all guest emails

Use the **Disable all booking emails to guests** toggle to turn off every guest email at once. This does not affect workflow emails.

## Manage individual email types

You can also enable or disable each email type separately. The available email types are:

| Email type              | What it sends                                                          |
| ----------------------- | ---------------------------------------------------------------------- |
| Confirmation            | Booking confirmation details sent when a meeting is booked             |
| Cancellation            | Notice that a booking has been cancelled                               |
| Rescheduled             | Updated details when a booking is rescheduled                          |
| Request                 | Notification that a booking has been requested and is awaiting approval |
| Host reassignment       | Notice that the meeting host has changed                               |
| Awaiting payment        | Reminder that payment is required to confirm the booking               |
| Reschedule request      | Request for the attendee to pick a new time                            |
| Location change         | Updated location details for an existing booking                       |
| Guest added             | Notification sent when a new guest is added to a booking               |
| Cal Video recording     | Download link for the Cal Video meeting recording                      |

<Note>
  When you disable the **Cal Video recording** email, only the attendee email is suppressed. The organizer still receives the recording download link.
</Note>

Each toggle requires confirmation before it takes effect. When disabling an email type, a confirmation dialog appears to make sure the change is intentional.
